# ShopEasy

A modern e-commerce web application built using React and Node.js.

 ## GitHub Repository
https://github.com/rbharathi7995/amazon-clone-react-ts

## Features

* Browse products
* Search products
* Add products to cart
* Update cart quantity
* Delete cart items
* Checkout page
* Delivery options
* Responsive design

## 📦 Installation

Clone the repository:

git clone https://github.com/rbharathi7995/amazon-clone-react-ts.git

Move into the project:

cd amazon-clone-react

Install backend dependencies:

cd ecommerce-backend
npm install

Install frontend dependencies:

cd ../ecommerce-project
npm install

## ▶️ Run the Project

Start the backend:

cd ecommerce-backend
npm run dev

Open a new terminal and start the frontend:

cd ecommerce-project
npm run dev

Open the local URL displayed in the terminal (for example, http://localhost:5173).
